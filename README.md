# id.140dev — портфолио

Сайт-портфолио фронтенд-разработчика id.140dev. Сделан как редакторский разворот: тёмная палитра, крупный итальянский курсив, моноширинный служебный голос, шейдерное поле на главной, плавные «створки» между маршрутами.

---

## 1 · Концепт коротко

Не SaaS-лендинг и не «hero + features + testimonials». Главная — это разворот с большим именем поверх медленного шейдера; работы лежат не сеткой, а **табличной разлинеёнкой** с крупными названиями; страница проекта строится вокруг **большой outline-цифры** номера работы и блоков описания (текст / стек / цитата / код).

Палитра: кость, пустота, железо, ржавчина, плюс редкая acid-зелень для статусов. Никаких градиентов, glassmorphism и rounded-2xl-shadow-xl.

Шрифты:

- `Instrument Serif` — итальянский курсив, дисплейный.
- `Inter Tight` — гротеск с font-stretch для заголовков.
- `JetBrains Mono` — служебный голос (метки, мета, навигация).

Шум живой: SVG-зерно сдвигается на пол-пикселя каждые 80 мс через `requestAnimationFrame`.

## 2 · Маршруты

```
/                — главная (имя + шейдер)
/work            — список работ (таблица, не сетка)
/work/[slug]     — страница проекта
/about           — обо мне
/contact         — контакт (адрес скрыт за hold-to-reveal от ботов)
```

Навигация — вертикальный список по правому краю с кодами `01–04` и хайр-линией активного пункта.

## 3 · Ключевые взаимодействия

- **Створки между маршрутами** (Barba-эквивалент для App Router): семь вертикальных полос закрывают вьюпорт `from: start`, после `router.push` уходят `from: edges`. См. `src/components/shell/Transition.tsx`.
- **Шейдерная плоскость** на главной: фрактальный шум + строчки развёртки + притяжение к курсору. Three.js подгружается лениво.
- **Кинетическая лента глифов** под именем: моноширинная строка, в которой раз в 220 мс случайный символ подменяется и возвращается через 90 мс.
- **Кастомный курсор** через `mix-blend-mode: difference` — точка моментально, кольцо отстаёт.
- **Список работ** — на hover строка чуть смещается и применяет `skewX(-2deg)`. Не fade.
- **Страница проекта** — outline-номер на 20vw въезжает с `xPercent: -8`, блоки приходят `y: 32 → 0`.
- **Контакт** — кнопка-удержание заполняется костью; при пересечении 50% надпись инвертируется через `mix-blend-mode: difference`.
- Везде уважается `prefers-reduced-motion`.

## 4 · Стек

- **Next.js 14** (app router, SSG для `/work/[slug]`)
- **Tailwind 3** — только утилитарный runner. Цвета и шрифты прокинуты через CSS-переменные, готовых компонентов нет, `corePlugins.container` отключён.
- **GSAP 3** + `ScrollTrigger` + `CustomEase` (изинги `arch` и `shutter`).
- **Three.js 0.166** — один `ShaderMaterial`, ленивая загрузка.
- Слой переходов написан вручную: Barba.js нативно сделан под классические MPA, поэтому в App Router тот же паттерн воспроизведён без самой библиотеки.

## 7 · Где менять контент

- Список проектов: `src/lib/data.ts` → массив `PROJECTS`. Каждый проект имеет `id`, `slug`, `title`, `year`, `client`, `role`, `status` (`live` / `private` / `wip`) и массив блоков (`text` / `spec` / `code` / `quote` / `image`).
- Имя, специализация, город: `src/lib/data.ts` → объект `META`.
- Тексты «обо мне»: `src/app/about/page.tsx`.
- Контакты (почта, телеграм, github): `src/app/contact/page.tsx`.
